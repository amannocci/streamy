# Use a plugin

## Date

The Streamy Date plugin provides components to manipulate date.

### Artifacts

```scala
val StreamyVersion = "0.13.0"

libraryDependencies ++= Seq(
  "io.techcode.streamy" %% "streamy-plugin-date" % StreamyVersion
)
```
